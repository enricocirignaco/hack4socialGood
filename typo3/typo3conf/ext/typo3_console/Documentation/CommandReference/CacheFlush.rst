===========
cache:flush
===========


**Flush all caches**

Flushes TYPO3 core caches first and after that, flushes caches from extensions.



Options
~~~~~~~

`--files-only`
   Only file caches are flushed

- Accept value: no
- Is value required: no
- Is multiple: no
- Default: false
