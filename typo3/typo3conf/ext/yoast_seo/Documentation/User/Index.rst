How to use
==========

Usage
-----
Your page will be analysed on readability and on the focus keyword you set. The analysis will give you tips to optimize
your content to have a better ranking in search engines.

Settings
--------
As an editor you can set some options.

Hide content analysis in Page module
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
In your user settings, you have the possibility on the tab Yoast SEO to hide the content analysis and snippet preview
in the Page module. Even if you have hidden the analysis in the page module, you can always view the analysis by the
Yoast SEO backend module.
